Privacy Policy

Single-Purpose Statement

The sole purpose of this extension is to help users download and manage all images on Ycut case detail pages with a single click, streamlining the manual, one-by-one image download process.

Reason for “downloads” Permission

To automatically download the selected image files into a user-specified local folder, avoiding the need for users to right-click and “Save as” each image individually.

Reason for “downloads.open” Permission

To automatically open the download folder once all images have finished downloading, allowing users to immediately view their downloaded files and improving convenience.

Reason for “scripting” Permission

To inject and execute content scripts that dynamically insert a “Download All Images” button into the case detail page, capture image URLs, and kick off the download workflow.

Reason for “storage” Permission

To save user preferences—such as the name of the last-used folder and other simple settings—so they can be automatically applied on subsequent uses, providing a smoother experience.

Reason for Remote Code Usage

This extension does not load or execute any remote code; all logic runs entirely on the user’s local machine.

Reason for Site Access Permission

Access is granted only to the single domain https://is.ycut.com.tw/is/case/detail/* for the purpose of retrieving image URLs and triggering downloads on that page; no other sites are accessed or their data collected.
